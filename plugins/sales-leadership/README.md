# plugins/sales-leadership

The sales-leadership plugin is the leader-shaped view over the six schema slots. It supports three modes: `VELOCITY` (TAM-velocity metrics for a segment, with a sensitivity note identifying which input metric moves overall velocity the most when shifted by 10%), `WIN_LOSS` (pattern refresh over closed-won and closed-lost cohorts, grouped by category labels from the controlled vocabularies in `schema/conversation_evidence`), and `BOARD_DELTA` (deterministic constraint-filter comparison of an opportunity list under two constraint sets, typically "current plan" vs "board ask"). The skill prompt is `SKILL.md`; the scorer is `board_vs_plan_scorer.py`. Every output is a **draft for reviewer judgment** and contains no customer, exec, or vendor names.

The `BOARD_DELTA` scorer is a **deterministic constraint filter, not a constrained optimizer**. It answers "which opportunities pass each set, and where do the deltas fall?" — not "what is the revenue-maximizing allocation subject to these constraints?" Real constrained optimization is a stretch goal, and the plugin says so if the operator asks for optimization. The plugin does not give financial or investment advice, does not commit to forecasts, and takes no external action without explicit user approval. Run the scorer standalone with `python plugins/sales-leadership/board_vs_plan_scorer.py` to see the demo with four synthetic opportunities and two constraint sets.
