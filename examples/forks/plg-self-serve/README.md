# plg-self-serve

Thin overlay for a product-led self-serve motion.

What changed vs. the base harness:

- Replaces the six enterprise account slots with product usage, activation, and
  expansion slots.
- Uses growth and customer-success shaped plugin defaults.
- Moves theory constants toward activation and expansion timing instead of
  consulting/implementation spend ratios.

This fork is a stub. It does not implement full agents.
