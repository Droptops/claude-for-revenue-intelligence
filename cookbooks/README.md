# cookbooks/

Cookbooks are end-to-end workflows that compose agents and plugins against the
active skill's schema. Each cookbook is intended to be readable as a worked
example, not packaged as a product.

| Cookbook | File |
|---|---|
| morning_dossier | `morning_dossier.md` |
| revenue_command_center | `revenue_command_center.md` |

The morning dossier is the headline workflow. The revenue command center
extends the same plugin set into a weekly inspection loop.

Any cookbook that observes external sites must be used in compliance with each
target site's `robots.txt` and Terms of Service.
