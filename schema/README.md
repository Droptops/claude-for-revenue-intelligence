# schema/

Schema slots are defined per skill. The base harness does not own a universal
revenue-intelligence schema; it loads the active skill and binds that skill's
schema contracts at install/runtime.

See `/skills/` for available installed skills, or run the cold-start interview
in `QUICKSTART.md` to select one for `CLAUDE.local.md`.

The original six-part reference schema now lives at:

`/skills/enterprise-account-based/schema/`
