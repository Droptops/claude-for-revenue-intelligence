# Morning Dossier Demo

Runs the local scorers over committed synthetic data and prints a
morning-dossier-shaped briefing.

```bash
python demo/morning_dossier_demo.py
```

The demo uses only stdlib Python and local files. No network calls, no CRM
writes, no external credentials. All account, opportunity, and persona values
are placeholders.

The output is a draft for reviewer judgment.
